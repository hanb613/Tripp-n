# Tripp-n

공공데이터포털(https://www.data.go.kr/) 을 활용하여, <br>
지역별 여행지 정보 및 계획, 커뮤니티, 통계 등 다양한 서비스를 제공하는 여행 정보 사이트

<br>  

## ⌚ 개발 기간
`2023.05.18 ~ 2023.05.25`

## 💡 주요 기능
- 메인화면
  - 메인화면
  - 국내 여행 차트
- 회원관리
  - 회원가입
  - 로그인
  - ID/PW 찾기
- 마이페이지
  - 회원 정보 수정
  - 관심 여행지
- 여행지 정보
  - 지역별 여행지 정보
- 커뮤니티
  - 게시글 조회 / 작성 / 수정 / 삭제
  - 게시글 댓글 조회 / 삭제
- 공지사항
  - 공지사항 조회
  - 공지사항 작상 (Admin Only)
- 지역별 날씨 조회
- 여행 차트
